## 📖 About

This repository documents my journey of learning **Docker** as part of my **DevOps Engineer roadmap**.

It includes:

- Docker fundamentals
- Images & Containers
- Dockerfile
- Volumes & Networking
- Docker Compose
- Docker Registry
- Docker Security
- Hands-on Practice Projects

The goal is to build a strong understanding of containerization through both theory and practical examples.

# 💻 Common Docker Commands

```bash
docker build -t myapp .

docker images

docker run -d -p 8080:80 myapp

docker ps

docker stop <container>

docker rm <container>

docker compose up -d

docker compose down

docker pull nginx

docker push username/image:v1
```
